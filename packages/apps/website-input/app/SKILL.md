# website-input

Use this app to manage shared first-party website attribution installs and the
canonical event ingest surface for a site.

## What this package owns

- installation creation and rotation
- canonical website-event ingest
- durable website-event storage
- installation-scoped deduplication
- operator-readable validation output

## What this package does not own

- ad-platform ingest
- backend outcome truth
- attribution scoring
- dashboard UI
