# website-input Workplans

This folder tracks the package-local work needed to support the website-input
operator and validation lanes.

Current focus:

- `WIB-006`: operator proof and validation lane
- `WIB-007`: companion pixels and tag ownership policy

Package-local workplan files:

- `wib-006-operator-proof-and-validation-lane.md`
- `wib-007-companion-pixels-and-tag-ownership-policy.md`

The docs-first rollout order is:

1. define the proof script and retained artifacts
2. define the companion pixel ownership matrix
3. validate custom-code installs against the proof script
4. validate Wix installs against the proof script
5. lock the one-shot website instrumentation checklist

These workplans should remain aligned with the umbrella board, not replace it.
