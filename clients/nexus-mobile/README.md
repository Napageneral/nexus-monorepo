# Nexus Mobile

This monorepo owns the native mobile client surfaces and their matching mobile device adapters.

## Layout

```text
NexusKit/
ios/
  app/
  adapter/
android/
  app/
  adapter/
```

## Commands

From this directory:

```bash
pnpm ios:open
pnpm ios:build
pnpm android:assemble
pnpm android:test
```
