# HCI-004E Hosted Manager-Worker Sandbox Execution And Evidence Bundle

## Goal

Prove hosted Dispatch manager/worker execution with sandboxed implementation
and validation, plus minimum evidence-bundle capture.

## Scope

- manager session orchestration
- implementation worker sandbox run
- validation worker or validation sandbox run
- evidence bundle capture with session and sandbox metadata

## Non-Goals

- PR delivery
- Jira/Slack closeout
- browser overlay

## Acceptance

1. manager and worker roles are both proven on hosted cleanroom infrastructure
2. sandbox execution metadata is captured alongside outputs
3. the evidence bundle is sufficient for later delivery and review tickets
