# RAGV-006 Real Slack Golden Journey Profile

## Goal

Make real Slack the primary golden-journey proof lane for manager delivery and
message-driven review scenarios.

## Scope

- define the real-Slack validation profile
- install the Slack adapter into the cleanroom
- materialize the named real Slack connection
- run the golden-journey Slack steps in the dedicated review channel
- capture receipts and the primary review artifact

## Acceptance

- Dispatch can execute the Slack golden journey in a Docker-backed cleanroom
  against a real Slack connection
- the review artifact proves real message delivery and follow-up behavior
- fake Slack is no longer the default ticket-level proof for this lane
