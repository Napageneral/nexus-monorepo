# RAGV-008 Real Git And Bitbucket Golden Journey Profile

## Goal

Extend the real-adapter golden journey to include repository-backed work proof
through Git and Bitbucket.

## Scope

- define the real repository profile requirements
- project the required Git and Bitbucket connections into the cleanroom
- prove representative real repository actions such as:
  - branch creation
  - commit or patch application
  - pull-request or equivalent review artifact creation
- bind those actions to the dedicated review repository namespace

## Acceptance

- the cleanroom golden journey can use real repository connections rather than
  fake or host-only stubs
- the proof bundle includes concrete repository receipts
- Git and Bitbucket become part of the same integrated review story as Slack
  and Jira
