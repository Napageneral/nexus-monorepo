# RAGV-007 Real Jira Golden Journey Profile

## Goal

Extend the real-adapter golden journey to include Jira-backed work proof.

## Scope

- define the real-Jira validation profile requirements
- install and project the Jira adapter and connection
- prove one or more representative real Jira actions:
  - issue create or update
  - ingest or backfill confirmation
  - reflected state inside the review package

## Acceptance

- the cleanroom golden journey can use the named real Jira connection
- the proof bundle captures real Jira receipts and the resulting product state
- Jira becomes a first-class part of the integrated ticket-review journey
