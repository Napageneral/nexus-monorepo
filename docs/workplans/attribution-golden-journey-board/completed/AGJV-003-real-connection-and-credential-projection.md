# AGJV-003 Real Connection And Credential Projection

## Goal

Project the real MoonSleep attribution connections into the cleanroom without
turning the proof lane into a pile of leaked environment variables.

## Scope

- named Meta Ads connection
- named Google Ads connection
- named TikTok Business connection
- named TikTok Display connection
- named Shopify connection
- credential references and minimum materialization for cleanroom use

## Acceptance

1. the cleanroom receives named connection records explicitly
2. credential projection is by reference and auditable
3. the proof lane can backfill and start monitors without manual shell
   topology residue
4. the resulting connection projection model is reusable for later validation
   profiles
