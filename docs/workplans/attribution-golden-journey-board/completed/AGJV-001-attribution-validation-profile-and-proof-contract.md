# AGJV-001 Attribution Validation Profile And Proof Contract

## Goal

Define the structured validation profile and human-shaped proof script for the
full attribution golden journey.

## Scope

- one stable attribution validation profile id
- one cleanroom-first execution contract
- one explicit pre-execution validation script
- one evidence contract for success and failure

## Acceptance

1. the validation profile names the required adapters, apps, connections, and
   website resource set explicitly
2. the proof contract says what the validating agent will do before execution
3. the evidence contract includes structured result, proof summary, logs,
   browser traces, and one primary review artifact
4. the profile is aligned to sandbox-managed fresh-server cleanroom doctrine
