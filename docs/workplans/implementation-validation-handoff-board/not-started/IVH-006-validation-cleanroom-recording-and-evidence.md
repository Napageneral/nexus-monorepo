# IVH-006 Validation Cleanroom Recording And Evidence

## Goal

Make the primary recording and proof bundle come from the validation cleanroom
that proved the selected candidate artifact.

## Scope

- capture whole-session recording from the validation cleanroom
- attach the candidate artifact id, validation packet, and proof summary to the
  review package
- retain the right receipts, logs, and media for success and failure
- make the review surface point to the validation recording first

## Acceptance

- the primary demo artifact for a Dispatch ticket comes from validation, not
  implementation
- the review package says which candidate artifact and profile were proven
- reviewers can watch one recording and trust it proves the implementation
  output under review
