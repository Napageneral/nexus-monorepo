# IVH-007 First Source-Snapshot-Backed Dogfood Ticket

## Goal

Close one real Dispatch dogfood ticket end to end using a source-snapshot
candidate artifact and a fresh validation cleanroom.

## Scope

- run one live ticket through implementing, candidate publication, validating,
  and reviewing
- use the approved validation script and selected profile
- prove the golden journey from the emitted source snapshot
- collect the validation recording and structured review receipts

## Acceptance

- one real ticket completes through Dispatch with source-snapshot-backed
  validation
- the validation recording proves the candidate artifact, not an implicit base
  ref
- the resulting review package is good enough for human signoff
