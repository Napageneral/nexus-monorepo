# AIP-007 Channel Source Breakdowns And Trajectories

## Goal

Port the most valuable channel and source breakdowns from MoonSleep into the
shared attribution app.

## Acceptance

1. paid, organic, referral, and direct channel/source breakdowns are available
2. the app supports trajectory-style views rather than only flat lists
3. the resulting reads are grounded in app-owned marts
4. the UI makes it easy to compare sources without needing the MoonSleep ops
   worker
