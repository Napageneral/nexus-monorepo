# DGR-005 Canonical Demo Recording Lifecycle And Artifact Ownership

## Goal

Make one recording the primary review artifact for the whole journey.

## Scope

- recording starts after preflight passes
- recording spans the whole orchestrated golden run
- recording ownership and artifact paths are canonical

## Acceptance

- one primary recording path is attached to the run
- the recording covers Slack, Jira, and forge phases in one session
- per-phase recordings, if retained, are secondary diagnostics
