# Nexus Telegram Adapter

This package is the shared Telegram adapter for Nex.

## Consumer SDK

Generated consumer SDK:

- `sdk/nexus-adapter-telegram-sdk-ts/`

Regenerate it with:

```bash
pnpm run sdk:generate
pnpm run sdk:build
```

## Validate

```bash
pnpm test
pnpm build
```

## Package

```bash
./scripts/package-release.sh
```
