# GlowBot Security Officer Designation

Effective Date: 2026-02-26

## Designation

Security Officer: Tyler (GlowBot operator/owner)

## Responsibilities

1. Own HIPAA safeguards and compliance execution across frontdoor, runtimes, and adapters.
2. Approve or reject changes that increase PHI exposure risk.
3. Own incident response decisions and breach notification execution.
4. Maintain risk assessment and security policy documents.
5. Ensure unresolved high-risk items are tracked with explicit mitigation and timeline.

## Delegation

1. Engineering implementation can be delegated.
2. Security sign-off and incident command remain with the designated Security Officer unless formally reassigned in this file.
