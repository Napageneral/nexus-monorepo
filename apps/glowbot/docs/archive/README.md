# GlowBot Archive

Archived material is retained for search and historical context only.

Current state:

- `legacy-specs/` contains superseded material from the pre-taxonomy documentation layout
- `workplans/` contains superseded execution notes, gap analyses, and handoff
  artifacts that are no longer part of the active GlowBot plan

This folder is intentionally outside the active spec/workplan/validation flow. If historical material needs to become active again, it should be promoted back into the correct artifact type rather than edited in place here.
