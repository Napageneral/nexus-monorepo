Capabilities

Semantic Code Search

Go beyond keyword matching. Spike understands code structure, function relationships, and data flow to find exactly what you need.

Natural Language Q&A

Ask questions in plain English. "Where is retry logic implemented?" "How does auth work?" Get answers with exact file and line references.

Deep Hydration

Spike walks every file, every branch, every commit you point it at. The oracle builds a complete map before answering a single question.

Private & Secure

Your code never leaves your tenant. Each workspace runs on isolated infrastructure with encrypted storage and scoped access controls.

GitHub Integration

Install the Spike GitHub App, select your repos, and hydrate in one click. Webhook-driven sync keeps your oracle current on every push.

Control Plane

Full API surface for jobs, sync status, ask history, and request forensics. Built-in inspector UI for debugging and RCA.

How it works
1

Sign in

One-click Google OAuth. Your account is created and your workspace is provisioned automatically.

2

Connect GitHub

Install the Spike GitHub App on your org or personal account. Select which repos to index.

3

Hydrate

Pick a branch and commit. Spike clones, walks every file, and builds the oracle's semantic map.

4

Ask

Type your question. The oracle dispatches across nodes, synthesizes evidence, and returns a grounded answer.
