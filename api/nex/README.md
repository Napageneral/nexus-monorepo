# Nex API

This directory contains the generated canonical OpenAPI artifact for the Nex runtime API.

Source ownership stays in the Nex runtime API layer under `nex/src/api/`, primarily:

- `nex/src/api/runtime-operations.ts`
- `nex/src/api/openapi/nex-contract.ts`
- `nex/src/api/http-runtime-api-routes.ts`
